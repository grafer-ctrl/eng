📖 PRESENT PERFECT

Digunakan untuk: kejadian yang telah terjadi, pengalaman, dan hasil.

📐 Rumus

(+) S + have/has + V3 + O

(-) S + have/has + not + V3 + O

(?) Have/Has + S + V3 + O?

📝 Contoh

(+) I have finished my homework.

Saya telah menyelesaikan PR saya.


(-) I have not finished my homework.

Saya belum menyelesaikan PR saya.


(?) Have you finished your homework?

Apakah kamu sudah menyelesaikan PR-mu?

🔥 Trik Cepat Hafal

1️⃣ Kata Kunci

already, yet, ever, never, just, recently

2️⃣ Have / Has

I/You/We/They → HAVE

He/She/It → HAS

3️⃣ Gunakan V3

eat → eaten
write → written
go → gone

🧠 Cara Cepat Paham

✅ Selalu pakai HAVE atau HAS

✅ Kata kerja bentuk V3

✅ Pertanyaan pakai Have/Has di depan

✅ Negatif = have/has + not
